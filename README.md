# 🚀 Modern Portfolio

![Portfolio Preview](https://raw.githubusercontent.com/MohammadAsher/Modern-Portfolio/main/public/images/preview.png)

A stunning, interactive 3D portfolio website built with modern web technologies. Features a immersive 3D character, smooth animations, and a professional design that showcases creative and technical skills.

---

## ✨ Features

- **🎭 3D Interactive Character** - Immersive 3D character with custom animations
- **🎨 Modern UI/UX** - Sleek, contemporary design with smooth transitions
- **📱 Fully Responsive** - Works perfectly on all devices
- **⚡ Smooth Animations** - Fluid animations powered by GSAP
- **🎯 Custom Cursor** - Interactive cursor with hover effects
- **🌐 Contact Section** - Easy way to get in touch
- **💼 Work Showcase** - Beautiful project portfolio display

---

## 🛠️ Technologies Used

| Category | Technologies |
|----------|-------------|
| **Frontend** | React, TypeScript, Vite |

## 🚦 Getting Started

Follow these steps to run the project locally:

### 1. Clone the Repository

```bash
git clone https://github.com/MohammadAsher/Modern-Portfolio.git
cd Modern-Portfolio
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Start Development Server

```bash
npm run dev
```

### 4. Build for Production

```bash
npm run build
```
